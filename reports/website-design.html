Main Navigation

Top-level pages linked in the header on every page:

  • Home
  • Gallery — Stars, Hikes, Waterfalls, Canyons (each links to its location pages)
  • Map
  • About
  • Contact

Each category page links to its individual location pages. Every page returns to Home via the logo.

Wire Frame Pages

  • Home — featured photo, intro, category links
  • Stars / Hikes / Waterfalls / Canyons — category intro and photo grid
  • Location page — photo, story, rating, back link
  • Map — pins for all locations
  • About — short bio
  • Contact — suggestion form

Interactivity (optional)

  • Lightbox for full-size photos
  • Contact form for spot suggestions
  • Interactive map with location pins

Design Guide

Layout / Storyboard

Hand sketch of the three main page layouts — home, category, and individual location:

In short:

  • Home — featured image at top, intro text, category links below
  • Category — title, short intro, responsive photo grid
  • Location — large image, story and rating side-by-side (stacked on mobile)

Color Palette

Pulled from the Mt. Rainier Skyline study in Report 01 — deep navy as the base, slate purple for depth, light gray for legible text, and warm gold as the single accent.
