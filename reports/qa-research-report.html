Quality Assurance in Web Development

1. What does Quality Assurance mean for website development?

QA means checking the website all the way through development to make sure it works right. This catches errors early so the final site is functional and reliable.

2. The two key principles: "Fit for Purpose" and "Right First Time"

"Fit for Purpose" means the product does what it is supposed to do. "Right First Time" means mistakes should be eliminated so the site works correctly from the start.

3. How do web standards relate to web quality?

Web standards are rules from the W3C for how to write code like HTML and CSS. Following them improves quality by making a site work the same across different browsers and devices.

4. What does Web Accessibility mean?

Web accessibility means building a site so people with disabilities can use it. It follows four principles called POUR: perceivable, operable, understandable, and robust.

Accessibility Steps for My PWP

First, I can add alt text to my images so screen readers can describe them. Second, I can use strong color contrast so the text is easy to read. Third, I can use clear headings and navigation so the pages are easy to move through.
